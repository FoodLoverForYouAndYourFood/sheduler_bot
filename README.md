TaskPair — минимальный Telegram-бот для трекинга задач. Хранит задачи в SQLite, даёт короткие команды и каждое утро/вечер шлёт дайджест.

## Требования и запуск
- Python 3.11+.
- Установка: `python -m venv .venv && .venv\\Scripts\\activate && pip install -r requirements.txt`.
- Скопируйте `config.example.json` в `config.json` и заполните свои данные.
- Установите токен бота: переменная окружения `BOT_TOKEN` (в `.env` или через `set/export`).
- Запуск: `python bot.py`.

## Настройки `config.json`
- `report_time`: время утреннего отчёта (`HH:MM`).
- `evening_report_time`: время вечернего отчёта (`HH:MM`).
- `timezone`: таймзона, например `Europe/Moscow`.
- `users`: стартовые пользователи (id Telegram, alias, name). Они будут добавлены в БД как админы.

## Команды
- `/add <текст>` — новая задача (выбор исполнителя, приоритет, дата, DoD).
- `/todo`, `/doing`, `/done` — списки по статусам.
- `/all` — все задачи.
- `/me` — задачи, назначенные на тебя.
- `/update <id> <todo|doing|done>` — сменить статус.
- `/deadline <id> <YYYY-MM-DD|clear>` — сменить дедлайн.
- `/report` — отправить отчёт сразу.
- `/cancel` — отменить ввод задачи.

### Админам
- `/add_user <id> <alias> <Имя>` — добавить пользователя.
- `/add_admin <id> <alias> <Имя>` — добавить администратора.

## Что такое DoD
Definition of Done (DoD) — список критериев, по которым задача считается завершённой. Примеры пунктов DoD:
- описана проверка или тест, который должен проходить;
- задеплоено в нужное окружение;
- обновлены документы/README/скриншоты;
- уведомлён заинтересованный стейкхолдер.

В диалоге `/add` бот спрашивает DoD после дедлайна. Можно ввести текст или `skip`, чтобы оставить поле пустым. DoD отображается в карточке задачи и в отчётах.
